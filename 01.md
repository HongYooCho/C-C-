## 정수형
- **char (1바이트)ㅡ** 문자를 담는 데 쓰임 (문자도 숫자 값을 가지고 있어서 정수형 자료형에 담을 수 있다.)
- **short (2바이트)**
- **long (4바이트)ㅡ** 2^31 ~ +2^31-1 사이의 값을 저장할 수 있다. 즉, 2^32 개 많큼 저장 가능.
- **long long (8바이트)ㅡ**
- **int (4바이트)ㅡ**

#### int 와 ‌long의 차이?
원래 int 는 시스템에 따라서 자동결정되는 친구. 그런데 요즘 거의 다 대체로 long 으로 결정됨. 그래서 요즘 int 는 그냥 4바이트.
